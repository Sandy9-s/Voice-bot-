# 🎙️ Real-Time Voice Chatbot System

## 📌 Overview
This project is a real-time voice chatbot that enables users to interact using speech. It captures audio input, converts it into text, processes the query intelligently, and responds with synthesized speech output. The system is fully browser-based and uses free technologies without requiring any API keys.

---

## 🎯 Features
- Speech-to-Text using Web Speech API  
- Intelligent query processing  
- Wikipedia integration for real-time information  
- Basic mathematical calculations  
- Conversational responses  
- Text-to-Speech output  
- Low latency interaction  
- Supports English and Telugu  

---

## 🔄 Voice Processing Pipeline
User Speech → Speech Recognition → Text Processing → Wikipedia / Logic Engine → Response Text → Speech Synthesis → Audio Output

---

## 🧰 Technology Stack
- HTML5  
- CSS3  
- JavaScript  
- Web Speech API (SpeechRecognition & SpeechSynthesis)  
- Wikipedia Public API

## ▶️ How to Run
1. Clone or download the project  
2. Open `index.html` in a browser (Google Chrome recommended)  
3. Allow microphone access  
4. Click on the microphone button  
5. Start speaking
